# MediBook

Healthcare management system built with React, TypeScript, and Vite.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

- React 18
- TypeScript
- Vite
- TailwindCSS
- Radix UI
- React Router
- React Hook Form
- Zod
- TanStack Query
