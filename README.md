# MediBook

MediBook is a comprehensive healthcare management system designed to streamline the interaction between patients, doctors, and administrators. The platform provides a modern, intuitive interface for managing appointments, medical records, and clinical consultations.

## Key Features

- **Patient Portal:** Appointment booking, doctor directory, and access to medical records.
- **Doctor Dashboard:** Consultation management, clinical notes, and digital prescription creation.
- **Admin Management:** Comprehensive user management, appointment oversight, and system audit logs.
- **Role-Based Access:** Secure authentication and authorization for different user types.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, Shadcn UI
- **State Management & Data Fetching:** TanStack Query (React Query)
- **Routing:** React Router
- **Form Handling:** React Hook Form, Zod

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```
